## 🚀 可以开始使用

### 快速开始
1. 按 `Ctrl+Shift+P` → `Developer: Reload Window`
2. 点击左侧数据库图标 🗄️
3. 点击 + 新建连接
4. 开始使用！

### 测试功能
1. 运行 `Berry DB: Run Tests` 命令
2. 运行 `Berry DB: Debug Connections` 命令
3. 测试右键菜单
4. 测试查询执行
