## 📝 使用指南

### 快捷键
- `Ctrl+Enter` - 执行查询
- `Ctrl+Shift+P` - 触发智能提示
- `Ctrl+R` - 重新执行
- `Ctrl+Shift+R` - 刷新缓存

### 如何启用智能提示
1. 连接数据库
2. 打开 SQL 文件
3. 开始输入，智能提示会自动显示

### 查看查询历史
1. 打开 Berry DB 视图
2. 点击 "Query History" 视图
3. 或使用命令 "Berry DB: Query History"
