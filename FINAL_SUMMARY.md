## 🚀 立即开始使用

### 步骤 1: 重新加载
```
按 Ctrl+Shift+P → Developer: Reload Window
```

### 步骤 2: 创建连接
1. 点击左侧数据库图标 🗄️
2. 点击 + 按钮
3. 选择数据库类型（推荐 SQLite）
4. 填写连接信息
5. 点击测试连接
6. 点击保存连接

### 步骤 3: 连接并使用
1. 点击连接节点连接数据库
2. 展开节点查看数据库结构
3. 右键点击测试菜单功能
4. 点击 📄 New Query 新建查询
5. 输入 SQL 并执行
6. 查看右侧面板的结果

## 🐛 问题排查

### 右键菜单不显示
1. 重新加载窗口
2. 运行 `Berry DB: Debug Connections`
3. 检查输出面板

### 连接失败
1. 检查数据库服务
2. 检查连接信息
3. 点击测试连接
4. 查看错误日志

### 查询结果不显示
1. 检查是否已连接
2. 检查 SQL 语法
3. 查看右侧面板
4. 重新执行查询
