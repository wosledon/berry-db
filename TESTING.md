# Berry DB 测试指南

## 重新加载扩展
按 `Ctrl+Shift+P` → `Developer: Reload Window`

## 测试步骤

### 1. 连接管理

#### 1.1 新建连接
1. 点击左侧数据库图标 🗄️
2. 点击 **+** 按钮
3. 选择数据库类型（例如 SQLite）
4. 填写连接信息：
   - 连接名称：`TestDB`
   - 文件路径：`C:\test.db`（SQLite）或相应的数据库信息
5. 点击 **测试连接** 验证连接
6. 点击 **保存连接**

#### 1.2 连接数据库
1. 点击连接节点（未连接状态）
2. 应该自动执行连接
3. 连接成功后，节点变为可展开状态

#### 1.3 断开连接
1. 点击连接节点旁边的 **插头图标**（inline 按钮）
2. 连接应该断开

#### 1.4 删除连接
1. **右键点击** 连接节点
2. 选择 **删除连接**
3. 确认删除

#### 1.5 编辑连接
1. **右键点击** 连接节点
2. 选择 **编辑连接**
3. 修改信息后保存

### 2. 调试工具

#### 2.1 查看连接调试信息
1. 按 `Ctrl+Shift+P`
2. 输入 `Berry DB: Debug Connections`
3. 查看输出面板中的连接信息

### 3. 查询功能

#### 3.1 新建查询
1. 点击左侧 **📄 New Query** 按钮
2. 或右键点击数据库节点 → **Open Query Editor**
3. 输入 SQL 语句

#### 3.2 执行查询
1. 在 SQL 文件中编写查询
2. 点击编辑器标题栏的 **运行** 按钮
3. 或选中部分 SQL 后执行
4. 结果应该显示在右侧面板

#### 3.3 查看表数据
1. 展开连接 → 数据库 → Tables
2. 右键点击表 → **View Data**
3. 数据应该显示在右侧面板

### 4. 表结构查看

#### 4.1 查看字段
1. 展开表节点
2. 点击 **Columns** 节点
3. 应该显示所有字段及其类型

#### 4.2 查看索引
1. 点击 **Indexes** 节点
2. 应该显示所有索引

#### 4.3 查看外键
1. 点击 **Foreign Keys** 节点
2. 应该显示所有外键约束

## 预期结果

### 连接节点
- 未连接：灰色插头图标，不可展开，点击执行连接
- 已连接：彩色插头图标，可展开，显示数据库内容

### 右键菜单
- 连接节点应该显示：
  - ✅ Connect / Disconnect（inline 按钮）
  - ✏️ Edit Connection
  - 🗑️ Delete Connection

### 查询结果
- 显示在右侧面板（ViewColumn.Two）
- DataGrip 风格的数据网格
- 包含工具栏、查询预览、状态栏

## 常见问题

### 右键菜单不显示
1. 检查是否重新加载了窗口
2. 运行 `Berry DB: Debug Connections` 命令
3. 检查输出面板中的 contextValue 是否正确

### 连接失败
1. 检查数据库服务是否运行
2. 检查连接信息是否正确
3. 查看输出面板中的错误信息

### 查询结果不显示
1. 检查是否已连接数据库
2. 检查 SQL 语法是否正确
3. 查看右侧面板是否打开
