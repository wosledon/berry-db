## 🚀 快速开始

1. **重新加载窗口**
   ```
   Ctrl+Shift+P → Developer: Reload Window
   ```

2. **创建连接**
   - 点击左侧数据库图标 🗄️
   - 点击 + 新建连接
   - 填写信息并保存

3. **使用功能**
   - 点击连接节点连接
   - 右键测试菜单功能
   - 点击 📄 新建查询
   - 执行 SQL 并查看结果
