- ✅ 新建连接（支持 SQLite/PostgreSQL/MySQL/Redis）
- ✅ 连接/断开连接
- ✅ 编辑连接
- ✅ 删除连接
- ✅ 测试连接
- ✅ 密码加密存储

### 2. 数据库浏览
- ✅ 树形视图展示连接
- ✅ 展示数据库/表/视图/存储过程
- ✅ 表详细信息（字段/索引/外键）
- ✅ Redis 键值浏览

### 3. 查询功能
- ✅ 新建 SQL 查询
- ✅ 执行查询（全文或选中部分）

### 6. 调试工具

- ✅ 输出面板显示连接信息

## 📦 技术栈

- TypeScript
- VSCode Extension API
- sql.js (SQLite)
- pg (PostgreSQL)
- mysql2 (MySQL)
- ioredis (Redis)

## 🚀 使用流程

1. **安装扩展**
2. **点击数据库图标**
3. **点击 + 新建连接**
4. **填写连接信息**
5. **点击连接节点连接数据库**
6. **展开查看数据库结构**
7. **点击 📄 新建查询**
8. **输入 SQL 并执行**
9. **查看右侧面板的结果**

## 🐛 已知限制

1. PostgreSQL 和 MySQL 需要安装相应的数据库服务
2. Redis 需要运行 Redis 服务器
3. 大数据集可能会影响性能
4. 某些高级 SQL 功能可能不支持
