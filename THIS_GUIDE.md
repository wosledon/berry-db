# Berry DB - 最终使用说明

## 🎉 项目已完成！

所有功能都已实现并测试通过。现在可以使用 Berry DB 进行数据库操作了。

## 🚀 快速开始

### 1. 重新加载扩展
```
按 Ctrl+Shift+P → Developer: Reload Window
```

### 2. 运行测试（可选）
```
按 Ctrl+Shift+P → Berry DB: Run Tests
```
查看输出面板确认所有测试通过。

### 3. 调试连接（可选）
```
按 Ctrl+Shift+P → Berry DB: Debug Connections
```
查看当前连接状态。

### 4. 创建第一个连接
1. 点击左侧 **数据库图标** 🗄️
2. 点击 **+** 按钮
3. 选择数据库类型（推荐 SQLite）
4. 填写信息：
   - 连接名称: `TestDB`
   - 文件路径: `C:\test.db`
5. 点击 **测试连接**
6. 点击 **保存连接**

### 5. 连接并使用
1. **点击连接节点** 连接数据库
2. **展开节点** 查看数据库结构
3. **右键点击** 连接节点测试菜单功能
4. **点击 📄 New Query** 新建查询
5. **输入 SQL** 并执行
6. **查看右侧面板** 的查询结果
